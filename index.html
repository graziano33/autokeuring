⚙️ Instellingen



Plak hier de URL van je Google Apps Script Web App. Zie de instructies in de README.

Hoe Google Sheets koppelen:

1. Open Google Sheets en maak een nieuw blad.
2. Ga naar Extensies → Apps Script.
3. Plak de code uit google-apps-script.js in het script.
4. Klik op Implementeren → Nieuw implementeren → Web App.
5. Stel toegang in op "Iedereen" en kopieer de URL.
6. Plak die URL hierboven.
